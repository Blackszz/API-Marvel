# Descrição do projeto 
  Eu, Guilherme Santos, juntamente com o Murilo Rangel, desenvolvemos um aplicativo utilizando uma API da Marvel. O aplicativo foi organizado em 2 telas, na tela inicial uma lista de diversos personagens da Marvel, já a segunda tela é a descrição do respectivo personagem escolhido, com os dados vindos da API, como o seu nome, descrição e etc.
